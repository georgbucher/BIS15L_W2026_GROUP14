# Data from: Vampire bats in Belize harbor multiple *Trypanosoma cruzi* genotypes: implications for parasite transmission at the wildlife–domestic–human interface

Dataset DOI: [10.5061/dryad.sj3tx96jz](https://doi.org/10.5061/dryad.sj3tx96jz)

## Description of the data and file structure

The data CSV includes PCR positivity data for *Trypanosoma cruzi* derived from two genetic markers as well as metadata on sampled vampire bats in Belize.

### Files and variables

#### File: Dunsmore_et_al_2026_PLOS_NTDs.csv

**Description:** 

##### Variables

* BZID: unique identifier for each individual bat (character)
* DI: unique identifier for each individual bat based on band number (character)
* spec: 4-letter bat species code (DERO = *Desmodus rotundus*)
* date: capture date (MM/DD/YYYY)
* year: capture year (YYYY)
* site: code for capture site (KK = Ka'Kabish, LAR = Lamanai Archeological Reserve)
* sx: binary code for sex (M = male, F = female)
* rp: code for reproductive status (N = non-reproductive, Y = reproductive, L = lactating, P = pregnant)
* age: code for age grouping (A = adult, JSA = juvenile or subadult)
* SatDNA: binary PCR result (0/1) for genomic satellite DNA
* kDNA: binary PCR result (0/1) for minicircle kinetoplast DNA
* tcruzi: binary PCR result for *T. cruzi* (SatDNA == 1 or kDNA == 1)
